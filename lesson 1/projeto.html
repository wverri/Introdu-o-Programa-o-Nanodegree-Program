Utilização de tag's

As tag's são utilizadas utilizando as palavras chave entre "<" e ">" por exemplo, para deixar uma palavra em negrito utilizar a tag <b> ou <strong>.

Exemplos de tag's

  • <b>texto</b> torna o 'texto' negrito
  • <em>text</em> torna o 'texto' itálico
  • <p>texto</p> Define um parágrafo. O conteúdo do elemento <p> é colocado em uma "caixa"
  • <br> Cria uma quebra de linha. "Enter"
  • <div>conteúdo</div> Cria um container em um bloco
  • <span>conteúdo</span> Cria um container inline

Atributos

Algumas tag's aceitam atributos que trazem alguma informação adicional ao navegador, como link's, imagens, vídeos, etc.

  • <a href="link">texto</a> para criação de um link:
  • Atalho para o google
  • <img src="link da imagem" alt="texto alternativo à imagem" style="largura,altura"> inserir imagem.

Elementos inline e em blocos.

Os elementos HTML podem ser tantoinline ou emblocos. Os elemntos em linha seguem uma mesma formatação se mantendo de forma constante, já no uso de elementos em blocos eles criam uma "caixa" em volta do conteúdo (texto, imagem, etc.)
